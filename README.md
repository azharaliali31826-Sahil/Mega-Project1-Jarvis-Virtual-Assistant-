1. Jarvis Voice Assistant

Jarvis is a Python-based voice assistant that can perform tasks using voice commands.
It listens to the user, processes the command, and responds using speech.



2. Features
- Voice recognition
- Text-to-speech response
- Open websites (Google, YouTube, etc.)
- Tell current time and date
- Answer basic questions
- Perform system tasks



3. Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- datetime
- webbrowser
- os
